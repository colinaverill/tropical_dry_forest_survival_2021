readme for Werden et al. 2021.

This repository reproduces all data construction, analysis and visualization for Werden et al. 2021 XXXX.

All raw data is stored in the subdirectory "data".

1._data_construction - this contains all code to manipulate raw data into analysis ready products, including estimation of above and belowground principle components.

2._data_analysis - This subdirectory contains code to fit models of species level survival rates.

3._data_visualization - this subdirectory contains code to generate all figures in the manuscript.

functions - this sub directory contains any custom functions used in data construction or analysis pipelines.
